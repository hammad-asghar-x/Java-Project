# 🧠 Student Mental Health & Academic Stress Tracker

A secure, web-based MVC application designed to help university students monitor their daily mental well-being and enable counselors to proactively identify and support at-risk students.

## 🚀 Features
- **Student Portal**: Log daily mood/stress (1-10), add notes, toggle anonymity, and view weekly trend charts.
- **Counselor Dashboard**: Monitor student lists, view individual histories, and resolve high-stress alerts (>7 for 3+ days).
- **Admin Panel**: Manage counselor accounts securely.

## 🛠️ Tech Stack
- **Backend**: Java 17+, Spring Boot 3.x, Spring Security, Spring Data JPA
- **Frontend**: Thymeleaf, Tailwind CSS, Chart.js
- **Database**: MySQL / PostgreSQL
- **Build & Version Control**: Maven, Git, GitHub

## 📋 Getting Started
1. Clone the repository: `git clone <repo-url>`
2. Configure `application.properties` with your local database credentials.
3. Build the project: `mvn clean install`
4. Run the application: `mvn spring-boot:run`
